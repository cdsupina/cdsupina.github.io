Inspiration

wifirover inspiration

Telememe was created for Uncommon Hacks 2016 which was a meme themed hackathon. In the weeks approaching the hackathon, I saw a video by Tested on youtube about creating a replica Ghostbusters trap. The main thing that caught my attention from this build was their use of e-cigarettes and LEDs to make a smoke effect. In the weeks leading up to the hackathon I learned how to achieve a similar effect as I demonstrate in a video on my youtube channel. The reason behind the Teletubby body was simply due to the absurdness being appropriate for the hackathon that I was entering, and also the idea of putting the e-cigarette in the antenna.

Idea

wifirover idea

We wanted the project to be a companion toy for young adults getting into making memes. There would be two parts to this project, the first would be the companion toy which would interact with the user. The second part of the project would be the mobile app which would allow the user to create memes and send them to the companion toy.

How it Works

wifirover idea

The user must first download the Telememe app on their phone and connect to the companion toy via bluetooth. Using the app, the user can generate a meme and send it to the companion to be analyzed. Depending on if the meme was good or not the companion will give the user a positive or negative response using its various methods of output.

How it was Built

wifirover idea

The app for generating and sending the memes was made using Android Studio. The companion was built inside of a Teletubby costume for a child stretched around a protein powder container. An Arduino was used to program the output from the companion. The e-cigarette and air pump were controlled using a relay. Lights and buzzers were also connected to the Arduino. The Arduino was connected to a bluetooth module that could recieve serial data from the connected phone. Depending on the data from the phone, the companion would go through a good or a bad output routine which consisted of blinking lights, buzzing a buzzer and blinking LEDs.
